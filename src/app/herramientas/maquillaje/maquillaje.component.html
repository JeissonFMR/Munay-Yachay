TIPS PARA EL MAESTRO

  • El maquillaje no necesariamente es una labor que la desarrolle un experto.
  • El primer paso para el maquillaje de carnaval tiene que ver con definir el tema a trabajar para el diseño, puede ser la identificación de animales, personajes fantásticos, la naturaleza.
  • Para la escogencia del motivo del maquillaje se puede partir de una lluvia de ideas a partir de los intereses de los niños.
  • Es necesaria la definición de la zona corporal que se va a maquillar, para ello la cinta de enmascarar o cinta de papel ayuda a realizar está demarcación.
  • Se debe partir con la aplicación de un fondo, generalmente se recomienda el uso de tonos claros como el blanco, esto con el fin de estandarizar el tono de piel teniendo en cuenta la diversidad de nuestros cuerpos.
  • La aplicación de la pintura no debe ser demasiada, esto con el fin de conseguir un secado más rápido.
  • Se inicia la aplicación con trazos lineales horizontales, es necesario dejar secar, para luego aplicar de manera vertical la pintura.
  • Es recomendable contar con todos los colores preparados para empezar la aplicación y verificar la calidad de las pinturas, sobre todo que estén elaboradas con elementos que no afecten la piel (hacer pruebas de reacción y firmeza).
  • Cuando se hace cambio de color es importante limpiar muy bien el pincel o cambiarlo.
  • Tener en cuenta que los colores claros se sobreponen sobre los más oscuros.
  • La pintura corporal requiere un disolvente, se recomienda usar agua para hacer este proceso.
  • El uso de plantillas en acetato, sirven de molde para hacer un gran número de maquillajes. Para ello, es necesario probar la plantilla en un papel antes de pasar al maquillaje en la piel.
  • Para hacer trazos a pulso es necesario probar estos trazos en un papel, para que se realice un trazo continuo y ágil.
  • Es recomendable el uso de un fijador para el maquillaje para la danza, para evitar su deterioro por el sudor, se puede usar una mezcla de agua con un poco de glicerina o laca, cuidando el contacto con los ojos.
  • Con el propósito de ser más efectivos en el uso de los materiales, se recomienda contar con los colores primarios como base para el maquillaje, pues se pueden mezclar para contar con otras gamas cromáticas.
