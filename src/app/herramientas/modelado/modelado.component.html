TIPS PARA EL MAESTRO

  • Para hacer varias réplicas de una pieza se requiere tener una base o molde, la elaboración del modelado le permitirá al maestro realizar varios elementos manteniendo el mismo diseño.
  • Para hacer el molde, se recomienda el uso de materiales durables como la plasticera, que es una mezcla de plastilina, parafina y cera de abeja.
  • Se pueden usar bandejas de aluminio para conservar la plasticera que servirá como materia prima para modelar
  • Este material es maleable, con el calor generado a través de la manipulación manual es posible dar la forma que se requiera.
  • El tema para realizar el modelado tiene que ir en conexión con la propuesta artística, generalmente se utilizan animales, personajes fantásticos o elementos representativos de la región.
  • Para empezar el paso de esculpir las figuras, se recomienda imprimir la imagen que se quiera modelar, definiendo el tamaño. La figura impresa sirve para calcarla en la plasticera para esculpirla en una sola dimensión, marcando los rasgos principales a través de un bisturí, punzón o paleta de madera.
  • La manera de dar volumen a la figura, para que sea bidimensional, se realiza con la fricción, el uso de herramientas y con la incorporación del material restante de la figura.
  • El molde es la base para realizar la figura con papel encolado, se debe aplicar una base de talco o vaselina para evitar la adherencia del papel con la plasticera.
